## 💡 Чему научился

- Prompt Object Pattern: промпт — это объект с `template + variables + rendered`
- Интерполяция через `/\{\{(\w+)\}\}/g` — достаточно для большинства случаев
- API route в Next.js 14 — `export async function POST(req: NextRequest)`
- Архитектура правильная с первого раза — важно доводить до конца цепочку
