# 📋 End-of-Session Ritual

> Запускай это в конце каждой рабочей сессии.
> Займёт 5 минут. Сделает тебя в 10 раз эффективнее.

---

## Шаг 1 — Обнови SESSION_LOG.md

Добавь запись:
```markdown
## YYYY-MM-DD — [название]
**Цель:** ...
**Сделал:** ...
**Не получилось:** ...
**Вывод:** ...
**Файлы:** ...
```

---

## Шаг 2 — Обнови MEMORY.md

- Добавь строку в таблицу «Что уже сделано»
- Обнови «Что пробовал / что сломалось» если нужно
- Добавь в «Чему научился» если что-то новое
- Обнови блок «Контекст для следующей сессии»

---

## Шаг 3 — Обнови GOALS.md

- Вычеркни выполненные задачи (`[x]`)
- Добавь новые идеи которые появились
- Перенеси выполненное в MEMORY.md

---

## Шаг 4 — Коммит

```bash
git add MEMORY.md SESSION_LOG.md GOALS.md
git commit -m "memory: update session YYYY-MM-DD"
git push
```

---

## Шаг 5 — Следующая сессия начинается с:

```
1. Открой MEMORY.md — вспомни кто ты и что делал
2. Открой GOALS.md — выбери следующую задачу
3. Начинай кодить
```

---

> **Почему это работает:**
> AI не имеет памяти между сессиями — но файлы в репо есть всегда.
> Ты строишь внешний мозг для себя и для AI который тебе помогает.
> Через 3 месяца у тебя будет полная история всего проекта.
